# Sistema de Gestión de Biblioteca Digital

Sistema de gestión de biblioteca desarrollado con arquitectura de microservicios usando Spring Boot, React y Docker.

# Patrones utilizados

- Patrón Api Gateway
- Patrón DTO + Services + Repository 

# Tecnologías

- Java + Spring Boot
- Base de datos MySQL
- React
- Docker
- Git

# Microservicios implementados

- Usuarios
- Libros
- Prestamos

Proyecto desarollado para Desarollo de software 3.
