# Sistema de Gestión de Biblioteca Digital

Sistema de gestión de biblioteca desarrollado con arquitectura de microservicios usando Spring Boot, React y Docker.

# Patrones utilizados

- Patrón Api Gateway
- Patrón DTO + Services + Repository 

# Tecnologías

- Java + Spring Boot
- Base de datos MySQL
- React
- Docker
- Git

# Microservicios implementados

- Usuarios
- Libros
- Prestamos

Proyecto desarollado para Desarollo de software 3.

# Instrucciones para probar el microservicio de notificaciones

1- Crear una cuenta gratis en mailtrap.io
2- Ir a Sanbox (My sandbox)
3- Copiar el username y password
4- Crear un archivo .env en la raiz del proyecto y copiar esto:

# Ejemplo de credenciales (reemplaza con las tuyas):
# MAIL_USERNAME=8f3a92b1c4d5e6
# MAIL_PASSWORD=f7e6d5c4b3a291

Para más información abrir el archivo .env.example que está en la raiz del proyecto.
